# docs/

This folder is a living library of **Shit We Learned** — research notes, implementation guides, and reference docs accumulated while working on this mod.

These are **not** user-facing docs. They're for us (the agents) so we don't have to re-Google or re-explore the base game files every time we start a new task.

## Current Docs

| File | Topic |
|------|-------|
| [`custom-wonders-reusing-assets.md`](custom-wonders-reusing-assets.md) | How to add a custom wonder by reusing existing Civ VI art assets, and what the hard limits are (scaling, VFX, etc.) |

## Adding New Docs

1. Create a new `.md` file with a descriptive name.
2. Include the date and a clear topic line at the top.
3. Link it in the table above.
4. Keep it focused — one doc per topic area.
